
# Interpretable ML Benchmark Framework

This repository provides a modular benchmarking pipeline integrating:

- Filter, wrapper, embedded, and metaheuristic feature selection
- XGBoost-based evaluation
- Feature Co-occurrence Network (FCN)
- PageRank-based feature integration
- Network visualisation

The framework demonstrates reproducible AI experimentation for structured and tabular datasets.

## Run

pip install -r requirements.txt
python run_pipeline.py
